# Mudah Belajar Ilmu Shorof

**Mudah Belajar Ilmu Shorof** is an EdTech platform designed to modernize and digitize Arabic language learning, with a specific focus on mastering **Ilmu Sharaf** (Arabic morphology). Developed in close collaboration with academic experts and university lecturers, the platform bridges traditional textbooks with digital interactive evaluations.

## ✨ Key Features

  * **Curriculum-Integrated Quizzes:** Interactive evaluation tools strategically placed at the end of each learning module to measure student comprehension.
  * **Physical-to-Digital Bridge:** Seamless access via **QR Codes** embedded in physical textbooks, allowing students to transition instantly from reading to digital testing.
  * **Academic Monitoring:** A comprehensive dashboard for educators to track student progress, identify learning gaps, and manage curriculum effectively.
  * **Fullstack Architecture:** A robust and scalable system designed to handle complex data relationships and real-time evaluations.

## 🛠 Tech Stack

  * **Framework:** Next.js (React)
  * **Database ORM:** Prisma
  * **Database:** PostgreSQL
  * **Styling:** Tailwind CSS
