# E-POLLING / E-BALLOT

A dedicated polling and balloting application designed for **Paskibra** competitions. This project serves as a specialized sub-module of the [E-Scoring](https://github.com/Rizki36/Escoring) ecosystem.

## 🚀 Key Features

  * **Real-time Statistics:** Visualize voting results and participant data instantly.
  * **Participant Management:** Dedicated interface for viewing and managing competition entrants.
  * **User-Friendly Interface:** Built with a modern aesthetic to ensure ease of use during high-pressure events.

## 🛠️ Tech Stack

  * **Target Platform:** Windows Desktop
  * **Language:** C\# (.NET Framework)
  * **UI Framework:** Bunifu UI Framework (for enhanced modern components)
  * **Database:** Microsoft Access (`.accdb`)

## 💻 Getting Started

To run the application directly without compiling, navigate to the following directory:

```bash
Hopeproject/bin/debug/hopeproject.exe
```
